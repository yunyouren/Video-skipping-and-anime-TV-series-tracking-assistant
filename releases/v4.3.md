# v4.3 Release Notes

## 中文

### 新增
- **每个收藏独立保存开关状态**：每个番剧可单独控制是否启用跳过，互不影响
- **白名单模式默认开启**：首次安装后弹出提示告知用户当前工作模式

### 移除
- **方案/预设功能**：原有的"自动匹配方案"功能已移除，被收藏专属设置和白名单模式替代
- **黑名单功能**：白名单模式下未收藏视频不会触发跳过，黑名单不再必要
- **保存按钮**：所有设置改为即时自动保存，无需手动点击

### 优化
- **收藏专属设置**：每个番剧独立保存片头/片尾跳过秒数、短视频保护阈值
- **自动保存**：所有数值输入框和开关均即时保存，改动即生效
- **弹窗精简**：移除冗余的预设管理区域，界面更清爽
- **代码精简**：减少约 450 行冗余代码，提升加载速度

### 兼容性
- 旧版本用户数据完全兼容，无 `autoSkipEnabled` 字段的收藏默认视为开启状态

## English

### Added
- **Per-favorite toggle state**: Each series independently remembers whether auto-skip is enabled for it
- **Whitelist mode notification**: First-time users see a notice explaining the default whitelist behavior

### Removed
- **Preset/Profile system**: Auto-matching presets replaced by per-favorite settings and whitelist mode
- **Site blacklist**: Redundant under whitelist mode — un-bookmarked videos are never skipped
- **Save button**: All settings now auto-save on change

### Improved
- **Per-series settings**: Intro/outro skip times and duration threshold saved independently per favorite
- **Instant auto-save**: Every input and toggle saves immediately — what you see is what you get
- **Cleaner popup UI**: Removed the preset management section, tighter layout
- **Leaner codebase**: ~450 lines removed for faster loading

### Backward Compatibility
- Existing user data is fully compatible; favorites without `autoSkipEnabled` default to enabled
