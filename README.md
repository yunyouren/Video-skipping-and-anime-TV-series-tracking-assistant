# 视频跳略与追番追剧助手 (Bilibili Skipper)

[![Manifest Version](https://img.shields.io/badge/Manifest-V3-blue)](https://developer.chrome.com/docs/extensions/mv3/intro/)

这是一个功能强大的 Chrome 浏览器扩展，专为 Bilibili 及其他视频网站设计。它能够智能跳过片头片尾、自动连播下一集，并提供便捷的追番收藏与进度管理功能，助你享受流畅的观影体验。

## ✨ 功能特性

*   **智能跳过**: 自动跳过视频片头和片尾，支持自定义跳过时长。
*   **自动连播**: 播放结束后自动点击“下一集”，无需手动操作。
*   **多场景预设**: 内置 B站、爱奇艺、腾讯视频等网站的预设配置，支持根据域名自动切换。
*   **追番收藏夹**: 一键收藏当前番剧，自动记录观看进度（集数与时间），随时通过扩展图标快速续看。
*   **快捷键控制**: 提供全局快捷键进行快进、快退，支持自定义按键组合（默认 `Shift + →` 快进，`Shift + ←` 快退）。
*   **智能识别**: 自动解析视频标题，提取番剧名称和集数信息。

## 🚀 安装指南

由于本项目尚未发布到 Chrome 应用商店，您需要通过“加载已解压的扩展程序”方式安装：

1.  下载本项目代码到本地，或者 Clone 本仓库：
    ```bash
    git clone https://github.com/your-username/bilibili-skipper.git
    ```
2.  打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
3.  打开右上角的 **“开发者模式”** 开关。
4.  点击左上角的 **“加载已解压的扩展程序”** 按钮。
5.  选择本项目所在的文件夹（包含 `manifest.json` 的目录）。
6.  安装完成！现在你应该能在浏览器工具栏看到插件图标。

## 📖 使用说明

### 1. 基础设置
点击浏览器右上角的插件图标打开控制面板：
*   **开启/关闭**: 通过顶部的开关一键启用或禁用跳略功能。
*   **跳过时间**: 设置片头（Intro）和片尾（Outro）的跳过秒数。
*   **自动连播**: 勾选“自动播放下一集”以启用连播功能。

### 2. 预设管理
插件支持保存多套配置方案（预设）：
*   在“预设方案”下拉框中选择不同配置。
*   可以为特定域名（如 `bilibili`、`iqiyi`）绑定预设，插件会自动识别并切换。

### 3. 追番收藏
*   在观看视频时，点击面板顶部的 **“+ 收藏当前”** 按钮。
*   插件会自动记录番剧名、当前集数和播放进度。
*   下次打开面板，点击列表中的条目即可直接跳转到上次观看的位置。
*   开启 **“自动更新进度”** 后，插件会实时更新你的观看记录。

### 4. 快捷键
默认快捷键如下（可在设置中修改）：
*   **快进**: `Shift` + `→`
*   **快退**: `Shift` + `←`

## 🛠️ 开发说明

本项目使用原生 JavaScript (ES6+) 开发，无需构建工具。

*   `manifest.json`: 扩展配置文件 (Manifest V3)。
*   `content.js`: 注入页面的核心逻辑，负责时间监控、DOM 操作和自动跳转。
*   `background.js`: 后台服务，负责脚本注入和生命周期管理。
*   `popup.html` / `popup.js`: 扩展弹出层界面及交互逻辑。
