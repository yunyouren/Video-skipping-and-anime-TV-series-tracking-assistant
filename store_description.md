# Edge 插件商店发布文案模板

## 1. 简短说明 (Short Description)
**限制：** 132 字符以内
**建议文案：**
智能跳过片头片尾，自动记录追番进度。支持 B站/爱奇艺/腾讯等主流视频网站，提供强大的收藏夹管理、云端备份与快捷键控制，是您追剧补番的效率神器。

---

## 2. 详细说明 (Detailed Description)
**建议文案：**

### 🚀 核心功能
**全能视频跳略与追番助手** 是一款专为追剧党打造的效率工具，解决重复操作痛点，让看剧更流畅。

#### ✨ 智能跳略 (Smart Skipping)
- **自动跳过片头/片尾**：可为不同网站设置专属跳过时长（支持精确到秒）。
- **完播自动重置**：看完后自动重置进度，方便二刷。
- **自动下一集**：连播模式下，片尾自动跳转，无需手动点击。

#### 📂 强大的收藏管理 (Favorites Manager)
- **跨平台追番**：无论是在 B站、爱奇艺还是其他网站，统一管理所有观看进度。
- **智能文件夹**：支持创建自定义文件夹（如“补番中”、“完结神作”），分类整理您的收藏。
- **进度防丢失**：自动记录观看时间点，支持“仅记录最大集数”模式，防止重温旧集数时覆盖最新进度。
- **数据备份**：支持一键导出/导入 JSON 备份，数据掌握在自己手中。

#### ⚡ 高效控制 (Power Controls)
- **全局快捷键**：支持自定义快进/快退快捷键（默认 Shift+→ / Shift+←）。
- **自定义规则**：遇到识别不准的番剧名？支持自定义正则规则修正标题。
- **自动匹配方案**：根据当前网站自动切换配置方案，无需反复手动调整。

### 🌍 支持站点
- 哔哩哔哩 (Bilibili) - 深度优化，支持合集/列表解析
- 爱奇艺 (iQIYI)
- 腾讯视频 (Tencent Video)
- 优酷 (Youku)
- 芒果TV (MGTV)
- 樱花动漫等第三方站点

### 🔒 隐私声明
本扩展仅在本地存储您的配置和观看记录，**绝不上传**任何用户隐私数据到云端服务器。所有权限仅用于实现视频控制和标题解析功能。

## 5. 常见问题 (Q&A)
**Q: 为什么某些网站无法跳过？**
A: 请检查是否在扩展的“预设方案”中添加了该网站的规则，或尝试调整跳过时长。

**Q: 收藏的数据会丢失吗？**
A: 数据存储在浏览器本地。建议定期使用“设置页-收藏管理”中的导出功能备份数据。
